# solum-flash-jig

This is a 3D model for a flashing jig for Solum electronic shelf labels.

It's made to work with pogo probes with receptacles. The receptacles fit into the jig and 

I used the following from Aliexpress:

* Pogo Receptacle - R50-3 (or P50-3W with wires attached)
* Pogo probes - P50-J1/P50-D2/P50-E2

The different numbered STL files are according to the `$fn = x` number used within the OpenSCAD script to achieve round corners & cylinders rather than jagged edges.
